Matematika SMA - Komposisi & Fungsi Invers

Pelajari konsep fungsi matematika dengan pendekatan visual dan interaktif.

Komposisi Fungsi

  • Definisi: (f g)(x) = f(g(x)), umumnya tidak komutatif.
  • Sifat asosiatif & identitas: I(x)=x.
  • Contoh: jika f(x)=2x+8, g(x)=4x-8 maka (fg)(x)=8x-8.

Contoh Interaktif

Diketahui f(x) = 2x + 3 dan g(x) = x - 1. Hitung (f g)(x):

1 (f g)(x) = f(g(x))
2 = f(x - 1)
3 = 2(x - 1) + 3
4 = 2x - 2 + 3
5 = 2x + 1

Fungsi Invers

  • Fungsi bijektif memiliki invers f sehingga f(f(x))=x.
  • Langkah umum: ganti f(x)=y, tukar x dan y, selesaikan untuk y, itulah f(x).
Latihan: Diketahui f(x)=x+1 dan (fg)(x)=3x+4. Tentukan g(4).

Contoh Interaktif

Diketahui f(x) = 3x - 2. Tentukan f(x):

1 y = 3x - 2
2 x = 3y - 2 (tukar x dan y)
3 x + 2 = 3y
4 y = (x + 2)/3

Jadi, f(x) = (x + 2)/3

Grafik Fungsi

  • Menggambar grafik fungsi linear, kuadrat, dan eksponensial
  • Transformasi grafik: translasi, refleksi, dan dilatasi
  • Analisis domain dan range dari grafik fungsi

Contoh Interaktif

Gambarkan grafik fungsi f(x) = x - 4x + 3:

1 Titik puncak: x = -b/2a = 4/2 = 2
2 f(2) = 4 - 8 + 3 = -1
3 Titik puncak: (2, -1)
4 Titik potong sumbu y: (0, 3)
5 Titik potong sumbu x: (1, 0) dan (3, 0)
